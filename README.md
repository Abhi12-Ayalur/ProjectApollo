# ProjectApollo
Designs/docs/software/PCB for Project Apollo (open source oxygen concentrator)

For the goals/motivation for this project please see the article in Helpful Engineering blog: https://www.helpfulengineering.org/projects-news/project-spotlight-apollo-oxygen-concentrator/ 

System Requirements:
Output medical grade Oxygen ideally at 93% or more
Flow rate of 1-20 liters per minute
(5 l/m being standard for oxygen therapy)
